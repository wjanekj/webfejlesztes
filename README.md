Leírás: 
	A weboldal témája a magyar lóversenyzés. A weboldal betekintést nyújt a lóversenyzés két féle típusába (galopp és ügető); az egyedüli magyar lóversenypályára, a Kincsem Parkra; valamint jegyfogalásra is lehetőségük van a weboldalra látogatóknak.

A saját JavaScript leírása:
A JavaScript kód a kincsem.html oldalon található. Egy interaktív családfát jelenít meg, amely Kincsemnek, a csodakancának a családfája a nagyszülőkig visszamenőleg. A js program lényege, hogy kattintásra előbukkanjanak az adott ló szülei, majd ha már nincs szükség rájuk, bezárhatóak legyenek.  

A használt egyedi fonttípus: Young Serif és Instrument Serif  
	és annak helye: a style.css első sorában lett importálva a két betűtípus.
